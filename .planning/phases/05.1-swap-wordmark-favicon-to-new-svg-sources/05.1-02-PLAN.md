Each `<path>` should retain only its `d="..."` attribute (and the multi-line whitespace/indentation inside the `d=` payload).

   ```svg
   z"/>
   ```

   ```svg
   M374.468658,881.000000
   ...
   z"/>
   ```

5. **Closing tag** — Preserve the `</svg>` closing tag at the end of the file.

The resulting file should be ~3-5 KB total (was 13 KB). The size shrink comes from dropping the verbose root attributes and per-path styling attributes (~1-2 KB), not from minifying path data.
