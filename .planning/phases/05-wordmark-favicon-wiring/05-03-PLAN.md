
   - **Line 3: apple-touch third.** `rel="apple-touch-icon"` (NO `type="..."` attr — per D's discretion in CONTEXT line 94 it is optional and not added). iOS keys off `rel`, not `type`.

3. Use the `absURL` pipe on every `href`, mirroring the project's existing `<link rel="stylesheet" href="{{ "css/style.css" | absURL }}">` precedent at `baseof.html` line 23. Do NOT hardcode `/favicon.ico` or `https://...` — let Hugo resolve against `baseURL`.

4. Do NOT add: a 4th `<link>` tag (e.g., `mstile-*`, `manifest`, `mask-icon`); a `<meta name="msapplication-...">` tag; any conditional `{{ if ... }}`; any `<title>`-related markup; any inline `<style>` or `<script>`.

5. The file is small and self-contained. Per CONTEXT § "specifics", treat it as fire-and-forget infrastructure — once shipped it shouldn't change unless the brand assets change.
